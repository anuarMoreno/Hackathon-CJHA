Personalizar assets

Cambia el aspecto del Breakout editando assets.js:

  • Pala: color o sprite propio.
  • Pelota: color, radio o imagen.
  • Bloques: paleta de colores por fila o sprites.
  • Fondo: color base o imagen de fondo.

Igual que en el Dino, el motor usa formas básicas y opcionalmente imágenes, perfectas para iterar rápido con Cursor.
