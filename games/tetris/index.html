Personalizar assets

Edita assets.js para cambiar el estilo del Tetris:

  • Colores de cada pieza (I, O, T, L, J, S, Z).
  • Tamaño de celda y bordes.
  • Fondos sólidos o con imagen.
  • Sprites opcionales para las celdas.

Todo el motor está separado de la apariencia, ideal para probar cambios rápidos con Cursor.
